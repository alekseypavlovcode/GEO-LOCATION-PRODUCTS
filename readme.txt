# Описание проекта
В проекте имеется код для хранения данных продукта с геолокациями. Реализовано получение геолокации товара с помощью сервисов.
Имеется также админка для управления товарами, геолокацими и её состояниями

Данные которые необходимо хранить о локации:
- `country_code` - код страны
- `city'- город
- `region_code` - код региона
- `disable`  - не активно
- `country`  - страна
- `region`  - регион
- `test` - тестовые данные
- `ip` - IP адрес

Эти данные находятся с помощью геолокатора.

Основные данные о товаре:
 - `product_categories` - Категории товара
 - `products` - Товар
